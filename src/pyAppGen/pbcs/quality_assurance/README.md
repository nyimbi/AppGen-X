# Quality Assurance PBC

This package implements a standalone Quality Assurance capability for AppGen-X applications. It owns inspection plans, sampling schemes, inspection results, holds, nonconformances, release proofs, control assertions, configuration, rules, parameters, AppGen-X event inbox/outbox evidence, and a workbench with forms, wizards, controls, and agent skills.

Supported runtime datastore backends remain PostgreSQL, MySQL, and MariaDB. The package-local SQLite repository is only a local standalone smoke/test harness.
