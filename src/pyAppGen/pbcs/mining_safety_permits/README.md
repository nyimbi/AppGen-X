# Mining Safety and Permits PBC

`mining_safety_permits` is a standalone AppGen-X PBC for mine permit-to-work, critical controls, and regulatory evidence.

It implements permit drafting/approval, isolation and lockout verification, confined-space gas tests, ground control holds, blast plans and re-entry, shift handover, incident precursor capture, high-potential escalation, regulatory evidence packs, UI forms/wizards/controls, and an AI assistant that refuses unsafe shortcuts while producing confirmation-gated CRUD previews.

The PBC uses PostgreSQL/MySQL/MariaDB backends, AppGen-X event contracts, owned tables only, and a composed-agent skill namespace.
