# MRP Engine PBC

This package implements a standalone Material Requirements Planning capability for AppGen-X applications. It owns BOMs, demand and inventory projections, MRP runs, material netting, pegging, planned orders, releases, controls, supply proofs, configuration, rules, parameters, AppGen-X event evidence, UI forms/wizards/controls, and assistant skills.

Supported runtime datastore backends remain PostgreSQL, MySQL, and MariaDB. The package-local SQLite repository is only a local standalone smoke/test harness.
