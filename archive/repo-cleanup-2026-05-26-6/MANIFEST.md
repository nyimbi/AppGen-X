# Repo Cleanup Archive - 2026-05-26 6

This archive records reversible cleanup from the active repository tree.

Moved out of the active tree:

- `src/**/__pycache__/` and `tests/**/__pycache__/` to `runtime-cache/`
  because they are generated interpreter bytecode caches and are ignored from
  normal source control.
- `appgen-frontend/public/` to `untracked-empty-dirs/appgen-frontend-public/`
  because it was an empty, untracked directory with no active assets.
- `appgen-frontend/src/assets/` to
  `untracked-empty-dirs/appgen-frontend-src-assets/` because it was an empty,
  untracked directory and the frontend source has no asset references.

The runtime cache payload is intentionally ignored to avoid committing
generated bytecode. This manifest keeps the cleanup decision auditable.
